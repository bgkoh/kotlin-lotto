# kotlin-lotto

# 기능 정리
- [x] 0원 이상의 구입 가격을 입력 받을 수 있다.
- [x] 로또 한장의 가격은 1,000원 이다.
- [x] 로또 갯수의 번호를 반환 한다.
- [x] 로또 번호는 6개이다.
- [x] 로또 번호는 1~45 범위를 가진다.
- [x] 당첨 번호와 보너스 번호를 입력 받을 수 있다.
- [x] 보너스 번호는 로또번호와 중복일 수 없다.
- [x] 로또 등수를 정할 수 있다.
- [x] 구매한 로또 번호와 개수를 출력한다.
- [x]당첨 통계를 반환 할 수 있다.
    - 3개 일치
    - 4개 일치
    - 5개 일치
    - 5개 일치 + 보너스 일치
    - 6개 일치
- [ ] 구입 금액 대비 총 수익률을 반환 할 수 있다.
